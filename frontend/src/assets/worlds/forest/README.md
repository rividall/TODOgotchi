## Expected contents

| Subdirectory | What goes here |
|---|---|
| `ground/` | 1-2 tileable grass textures for the `TilingSprite` floor |
| `decorations/` | Static sprites scattered on the field — trees, rocks, flowers, bushes, stumps |
| `creatures/` | Poring character art per tier — `poring-seed.png`, `poring-happy.png`, `poring-chubby.png`, `poring-ripe.png`. Alternative: a single `poring.riv` Rive file for state-machine-driven animation. |

## Recommended Kenney picks

- **Primary source**: [Nature Kit](https://kenney.nl/assets/nature-kit) — trees, rocks, flowers, terrain
- **Bonus**: [Foliage Sprites](https://kenney-assets.itch.io/foliage-sprites) — higher-detail plants
- **Alt style**: [Sprout Lands](https://cupnooble.itch.io/sprout-lands-asset-pack) ($5) if you want a cohesive painterly-pixel vibe instead of Kenney's low-poly look
